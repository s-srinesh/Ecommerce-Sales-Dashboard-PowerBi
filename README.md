## Data Cleaning & Transformation

Before building the dashboard, the dataset was prepared and validated using Microsoft Excel to ensure data accuracy and consistency for analysis.

### Data Preparation Steps

* Inspected the dataset for missing values and data inconsistencies.
* Verified data types for sales, profit, quantity, and categorical fields.
* Standardized column formatting to maintain consistency.
* Validated sales, profit, and quantity records for accuracy.
* Structured and organized the dataset for Power BI import.
* Established relationships between Orders and Details tables using Order ID.
* Performed data modeling to support KPI calculations and reporting.

### Data Modeling

* Created relationships between Orders and Details tables.
* Optimized the data model for accurate reporting and analysis.
* Implemented DAX measures to calculate key business metrics.
* Applied filters and slicers for interactive dashboard analysis.

### DAX Measures Created

* Total Sales
* Total Profit
* Total Quantity
* Average Order Value (AOV)

### Business Intelligence Objectives

* Analyze sales performance across different states.
* Identify top-performing categories and sub-categories.
* Monitor monthly profit trends.
* Evaluate customer purchasing behavior.
* Analyze payment method preferences.
* Generate actionable business insights for decision-making.
